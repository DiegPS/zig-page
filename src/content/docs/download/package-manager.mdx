---
title: Install Zig from a package manager
description: Install Zig from a package manager
---

Note: Official builds of master branch and releases are
[available for download](/download/).

When using a package manager, it is best to use a tagged release rather than
using an option to install a development version.

## Alpine Linux

```sh
apk add zig --repository=https://dl-cdn.alpinelinux.org/alpine/edge/community
```

## Arch Linux

```sh
pacman -S zig
```

## DragonFlyBSD (ravenports)

```sh
ravensw install zig-single-standard
```

## Fedora

```sh
dnf install zig
```

## Fedora Silverblue

```sh
rpm-ostree install zig
```

## FreeBSD

```sh
pkg install lang/zig
```

## Gentoo

```sh
# Building from sources
emerge -av dev-lang/zig
# Official ziglang.org static build
emerge -av dev-lang/zig-bin
```

## Homebrew

```sh
brew install zig
```

## MacPorts

```sh
port install zig
```

## Mise

```sh
mise install zig
```


## NixOS

Rather than installing development binaries globally, create a `shell.nix` for your project:

```nix
# shell.nix
let
  pkgs = import <nixpkgs> {};
in
  pkgs.mkShell {
    packages = [
      pkgs.zig
      # other deps here
    ];
  }
```

Then run `nix-shell` to enter a development shell with zig available.

## Ubuntu (snap)
Stable:
```sh
snap install zig --classic --beta
```

## Void Linux

```sh
xbps-install -Su zig
```

## Windows (winget)

```sh
winget install zig.zig
```

## Windows (choco)

```sh
choco install zig
```

## Windows (scoop)

```sh
scoop install zig
```

## Termux

```sh
pkg i zig
```
